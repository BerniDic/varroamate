ℹ️ How inspections & mite counts sync
▾
📱
VarroaMate (VM) is the source of truth for mite counts. Counts entered via the green field button or red regular entry button in VM are stored in the mite_counts table and appear here as 🔬 3/300 pills on the matching inspection.
🔍
VarroaMate Productivity (VMP) adds a Mite Count section to the inspection modal. If you enter a count here, it is written directly to VM's mite_counts table and becomes visible in VM immediately — no duplication, same data.
🏷️
Entries made in VMP are tagged with a VMP badge so you can tell them apart from VM entries at a glance.
🔗
A mite count pill appears on an inspection when a count exists for the same hive within ±1 day of the inspection date — whether that count was entered in VM or VMP.
⚠️
AFB and EFB are nationally notifiable pests. If you select these in the disease observations, VMP will show a biosecurity alert with a link to your state authority. Always report positive detections promptly.
